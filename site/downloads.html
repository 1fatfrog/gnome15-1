Source

You have two options for obtaining the source. Either check out the trunk branch from the Launchpad project, or download the source :-

  • gnome15-0.3.1.tar.gz Gnome15 core and all plugins except Impulse15.

The Impulse15 plugin contains native portions and is available separately :-

  • gnome15-impulse15-0.0.3.tar.gz Impluse15 plugin and native library.

Pre-requisites

You will need all of these to compile or run Gnome15. If you use a package for your distribution, these will be downloaded for you. If you wish to compile from source, you will have to obtain and install them yourself.

  • GNOME - The free desktop.
  • Python - Programming language. Gnome15 was developed with Python 2.6
  • PyGTK - Python GTK bindings (including Python GNOME bindings, Python GObject bindings, RSVG bindings).
  • DBUS - Python bindings.
  • LXML - XML parser.
  • Python Cairo - Python bindings for the Cairo graphics library. Note, for best performance on the G19 you will need at least Cairo version 1.8.3.
  • PIL - Python Image Library.
  • Python X Library - Interface for Python to the X11 Protocol.

The following libraries are optional, but should be installed for the best experience.

  • setproctitle - Allows a process to change its title.
  • Python Appindicator - Allows Gnome15 to add an indicator (replaces applet).
  • Python WNCK - Python bindings for the WNCK library (must be installed when BAMF is not available for automatic macro profile switching).

The plugins have some further dependencies

  • evolution-python - Python Evolution bindings. Used for Calendar plugin.
  • vobject - Python vCard and vCalendar parser and generator. Used for Calendar plugin.
  • PyAlsa - Python Alsa bindings. Used to monitor mixer events for volume plugin.
  • MPlayer - Movie player. Used by the experimental Video plugin.
  • FFTW3 - Fourier transform library used by the Impulse15 plugin.
  • Pulse - Sound server. Required by Impulse15 plugin.

Finally, at run-time you will need one of the following depending on your hardware:-

  • G19D - Daemon for G19 keyboard.
  • g15daemon - Daemon for G15 and G13 keyboards or Z-10 speakers.
  • If you are brave, you might considering trying the patched kernel. This is currently completely working except for the LCD backlight control on the G19. The G15 support is about 50% complete (keyboard macros and controls work, LCD does not). G13 support has not been started.
