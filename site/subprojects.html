Summary

These are small sub-projects that Gnome15 depends on.

  • lgsetled - Sets LED brightness for kernel driver
  • g19d - Adds support for G19 (user space daemon)
  • Kernel Driver - Effort to get Gnome15 to use the Logitech G keyboard kernel drivers.
