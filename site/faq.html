FAQ

General Questions

What keyboard models are currently supported by Gnome15?

Currently only the Logitech G15 (v1) and Logitech G19 are supported (these are the two keyboards I have access to). Support for the other keyboards on the range will currently depend on support provided by g15daemon. There are patches available for at least the G13, although you are on your own here and I cannot guarantee it will work correctly in Gnome15. Efforts will soon begin to add support for other hardware.

Why does indicator menu not show the current screens on Ubuntu 10.4, yet it does on Ubuntu 10.10

Because screens come and go (e.g. with enabling / disabling of plugins), the indicator menu must be able to rebuilt at any time. Unfortunately, this is not possible in Lucid. A work-around may be available in a future release. As an alternative, you could use the applet version instead.

G15 Questions

Why is text not appearing on the LCD? Or why is the text hard to read?

To get the best results, the G15 driver requires that fixed size bitmap fonts are used to display text (currently the 'Fixed' font is used). Recent versions of Ubuntu at least disable these fonts by default. The Ubuntu gnome15 package tries to enable them for you, but for other distributions you will have to do this yourself. See this post for more information and a possible solution.

G19 Questions

Why does Gnome15 use so much CPU when drawing the screen on the G19?

You will see this if you are using a version of Cairo less than 1.8.3. This is because the 16 bit image format that that G19 requires was disabled in Cairo for a number of years and only recently re-introduced. Without this native support, Gnome15 has convert the image itself which is currently very slow.

The solution, ensure you are using at least Cairo 1.8.3.
